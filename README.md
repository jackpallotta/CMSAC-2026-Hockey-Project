# CMSAC-2026-Hockey-Project
# Leverage Score Steps
Step 1: run the ‘pbp.R’ script to scrape the NHL public play-by-play into the console and clean the data.
Step 2: run the ‘schedule.R’ script to scrape the NHL season schedule into the console and clean the data.
Step 3: run the ‘players.R’ script to scrape NHL player bios into the console and clean the data.
Step 4: run the ‘gameLogs.R’ script to scrape NHL player career totals into the console and clean the data.
Step 5: run the ‘rosters.R’ script to scrape the NHL rosters by game into the console and clean the data.
Step 6: run the ‘variables.R’ script to build a clean, mirrored event-level faceoff dataset for modeling.
Step 7: run the ‘leverage score.R’ script to run the models and calculate a leverage score for every faceoff.
Step 8: run the ‘wp model.R’ and ‘xG model.R’ scripts to compare modeling techniques for goal value and goal likelihood within 5 seconds of the faceoff.
Step 9: run the ‘leverage score analysis.R’ script to split the leverage scores into natural clusters and create data visualizations.
